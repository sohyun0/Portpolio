기업 이미지와 브랜딩을 인터렉션으로 표현되는 사이트를 구현하는 것이 핵심이였습니다.

첫 번째 요소로 메인페이지의 인트로 영역에서는 gsap.js의 timeline 기능을 활용하여, 스크롤 동작에 따라 톱니바퀴가 동적으로 회전하고, 이후에 텍스트가 부드럽게 노출되는 애니메이션을 구현했습니다.

컨텐츠 영역은 wow.js를 활용하여 사용자가 스크롤할 때, 컨텐츠가 순차적으로 나타나면서 동적인 느낌을 제공함과 동시에 많은 양의 정보가 아닌 최소한의 컨텐츠를 노출시켜 모든 내용을 다 볼 수 있도록 사용자 경험을 향상시켰습니다.

두 번째로, 클리닝 서비스의 Before와 After 이미지를 명확하게 비교할 수 있는 기능을 구현했습니다. 이를 위해 input 요소의 range를 활용하여 반반 슬라이드를 개발하였습니다.

마지막으로, 모바일 환경에서 서브 페이지의 사용자 경험을 개선하기 위해 드롭다운 탭을 구현했습니다. 화면 크기가 작아질 때, 탭이 자동으로 드롭다운 형태로 변환되어 사용자가 쉽게 이동할 수 있도록 했습니다.

그 결과 메인페이지는 사용자에게 기업 이미지를 시각적으로 강조하여 전달하는 역할, Before&After 슬라이드는 기업 브랜딩을 강화 할 수 있는 효과를 줍니다. 기업의 온라인 존재감을 향상시키고 고객과의 상호작용을 개선하는 데 기여했습니다.

또한 오픈 전 급작스런 디자인 변경 및 컨텐츠 미수급등 이슈가 있었으나 개발된 컴포넌트를 재사용하여 추가 작업을 최소한으로 유지하는 목표를 세운 후 원활한 커뮤니케이션과 효율적인 리소스 활용을 통해 프로젝트를 기한 내 성공적으로 완료했습니다.
