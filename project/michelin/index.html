해당 프로젝트에서는 스마트스토어와 비슷한 UI/UX를 구현하는 것이 핵심이였습니다.

스마트스토어를 분석 하여 검색 기능 UX 최적화, 필터 UI 차별화, 구매 버튼의 반응형 배치, 날짜 선택 UX 개선, 메인 비주얼과 상품 슬라이드 구현, 그리고 브랜드 특성을 반영한 페이지네이션 디자인을 통해 사용자의 편의성과 브랜드 경험을 극대화하는 데 초점을 맞추었습니다.

1. 디바이스별 최적화된 검색 경험을 제공 하기 위해 PC와 모바일의 특성에 맞춰 다르게 설계하여 최적의 사용자 경험을 제공했습니다. PC 환경에서는 키보드와 마우스를 활용하여 빠른 검색 및 페이지 탐색이 가능하도록 구성하였으며, 모바일 환경에서는 터치 인터페이스에 최적화된 검색 UX를 적용하여 손쉬운 조작과 가시성을 확보했습니다.
또한, 자동완성 기능을 추가하여 사용자가 입력하는 동안 관련된 추천 검색어를 실시간으로 제공함으로써 검색의 효율성을 극대화했습니다. 검색 결과 화면에서는 사용자가 원하는 항목을 빠르게 선택할 수 있도록 UI를 구성하여 검색 결과의 가독성을 향상시켰습니다.

2. 리스트 검색 필터(추천순/가격순/판매량순)의 UI는 넓은 화면(PC)에서는 라디오 버튼 형식, 좁은 화면(모바일)에서는 셀렉트 박스 형식으로 구현하여 화면 크기에 따른 공간 활용성을 극대화했습니다. 이를 통해 사용자가 최적의 환경에서 직관적으로 필터를 선택할 수 있도록 했습니다.

3. 메인 액션 버튼(구매 버튼)의 경우 디바이스별 사용자 경험을 고려하여 배치 방식에 차별화를 두었습니다. 넓은 화면(PC)에서는 사이드에 배치하여 화면 내 흐름을 유지하면서도 사용자의 시선을 자연스럽게 유도할 수 있도록 했고, 좁은 화면(모바일)에서는 플로팅 버튼 + 옵션 레이어 팝업 방식을 적용하여 한 손 조작이 용이하면서도 즉각적인 구매가 가능하도록 UX를 개선했습니다.

4. UX를 고려한 Datepicker 기능은 jQuery UI 기반의 Datepicker를 활용하였고, 인풋 클릭 시 3~7 영업일만 선택 가능하도록 구현함으로써 불필요한 입력 오류를 방지하고, 사용자 경험을 단순화했습니다. 이는 UX에서 중요한 직관적인 인터랙션을 고려한 설계입니다.

5. Swiper.js를 활용한 메인 비주얼 슬라이드의 페이지네이션 UI는 미쉐린 브랜드의 아이덴티티를 반영하여 타이어 형태의 버튼으로 커스터마이징했습니다. 이는 단순한 디자인 요소를 넘어 브랜드 정체성과 사용자 경험을 강화하는 인터랙티브한 UI 설계로, 브랜드의 고유한 아이덴티티를 직관적으로 전달하는 효과를 주었습니다.

이번 프로젝트의 UI/UX 설계는 단순한 기능 구현을 넘어 사용자의 행동 패턴과 브랜드 아이덴티티를 조화롭게 연결한 산출물이 만들어졌고, 디바이스별 차별화된 경험을 제공하기 위해 많은 UX를 구현해볼 수 있는 기회가 되었습니다.
