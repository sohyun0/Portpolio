Color

  • 1번째 팔레트는 Primary 와 Secondary
  • 2번째 팔레트는 Grayscale
  • 3번째 팔레트는 투명도로 이루어져 있는 Neutral
  • 4번째 팔레트는 4가지 타입이 있는 Functional
  • 클래스는 [type]-[color][number]의 구성으로 되어있습니다.

Grid System

  • 그리드 시스템의 기준은 Max width: 1600px , Gutter: 24px, Margin: 20px, Columns: 12cols 을 기준으로 합니다
  • 상위 클래스는 flex-box , 하위 클래스는 grid-item-[number]의 조합으로 이루어져있습니다.
  • 하위 클래스의 grid-item은 column을 조합해서 사용 할 수 있습니다.
  • grid-item-12는 단독으로 사용할 수도 있으며 , grid-item-12-[number]의 형태로 다양한 그리드를 만들 수도 있습니다.

Type Columns Position Class example Description

responsive - parent element flex-box respon 반응형 레이아웃의 컨테이너
no margin columns 2 | 3 | 4 | 5 | 6 | 10 | 12 parent element flex-box col[Option-Columns] 여백이 없는 레이아웃의 컨테이너 및 아이템 너비 설정
* 자식 컨텐츠에게는 별도의 클래스를 부여하지 않아도 됨
grid-item 1 | 2 | 3 | 4 | 6 | 12 child element grid-item[Option-Columns] 일반 그리드의 아이템
12 column grid 2 | 3 | 4 | 5 | 7 | 8 | 9 | 10 | 11 child element grid-item-12-[Option-Columns] 12 그리드의 아이템
단 , grid-item-12-1 은 grid-item-12 와 동일하고
grid-item-12-12는grid-item-1 와 동일하기 때문에
별도의 클래스를 생성하지 않았습니다.

Icongraphy

  • 모든 아이콘의 기본속성은 display:inline-block입니다.
  • 별도의 margin 클래스를 사용하지 않는 경우 중앙정렬 됩니다.
  • 기본 아이콘의 크기는 24px을 기준으로 합니다.
  • 아이콘 사용 시 ic ic-[size] [name] [background-color]을 한 묶음의 형태로 사용합니다.
  • 아이콘을 뜻하는 ic와 아이콘의 사이즈, 아이콘의 명 세가지 중 하나라도 없으면 정상적인 형태가 나오지 않습니다.
  • 아이콘의 컬러는 background-color로 지정할 수 있으며 기본색상은 #000 (black)입니다.

sm | md | lg
xl | xl2 | xl3 		bg-[color name][color number] ic ic-md arrow-top bg-white-88 아이콘 설정

Spacing

  • 그리드 시스템에서 사용하는 여백은 24px을 사용합니다.
  • 여백은 속성[m/p], 방향[t/b/ l/r], 수치[0~40]를 혼합한 상태로 구성되어 있습니다. 단, 수치는 2의 배수로 구성되어 있습니다.
  • m20, mt20, p20, pb20와 같은 형태로 조합하여 사용할 수 있습니다.

Option

Attribute Class example Number Description Attribute Class example Number Description
margin m 0~40 중 0을 포함한
2의 배수 		상하 좌우 바깥 여백 padding p 0~40 중 0을 포함한
2의 배수 		상하 좌우 안쪽 여백
margin top mt 상향 바깥 여백 padding top pt 상향 안쪽 여백
margin bottom mb 하향 바깥 여백 padding bottom pb 하향 안쪽 여백

margin right mr 우향 바깥 여백 padding right pr 우향 안쪽 여백

Typography

  • 기본 폰트 사이즈는 16px을 100%로 사용하여 1rem은 16px 로 적용합니다.
  • 폰트는 fs-[size]-[lineheight]의 조합으로 이루어져 있습니다.
  • 폰트의 굵기는 fw-[number] 의 조합으로 이루어져 있습니다.
  • 모든 태그의 기본 속성은 font-size : 16px, line-height : 1 , color : gray-900, font-weight : 400 입니다.
