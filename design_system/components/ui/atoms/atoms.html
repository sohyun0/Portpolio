Atomic Design - Atoms

Button

  • btn + btn-[size] [type] | [state]의 조합으로 이루어져 있습니다.
  • [size] [type] 은 필수로 지정해야 하고, [state]는 지정하지 않으면 활성화 상태로 적용됩니다.
  • component에 포함되지 않은 스타일은 아래의 방법으로 제작할 수 있고 별도의 상태별 스타일이 필요할 시 atoms.scss에서 작성 부탁드립니다.

Option

Type Style Size Class example Description

x-sm | sm | md | lg | xl 		class="btn btn-md fc-primary100 bg-primary500" 버튼 배경 스타일 설정 및 아이콘 적용

단 , 아이콘 적용 시 버튼 내부에 배치

Text only text x-sm | sm | md | lg | xl | full class="btn btn-md fc-primary500 text" 텍스트 버튼 스타일 설정 ( 내부여백 삭제 )

Example

<!-- default => background-color -->
<button type="button" class="btn btn-md bg-primary-500">Button</button>

<!-- line type => border-color  -->
<button type="button" class="btn btn-md bd-primary-500">Button</button>

<!-- text type => text fc-color  -->
<button type="button" class="btn btn-md text fc-primary-500">Button</button>

<!-- icon type => icon tag -->
<button type="button" class="btn btn-md bd-primary-500">
  <span class="ic ic-md arrow-top"></span>
</button>

<!-- full size button -->
<button type="button" class="btn btn-md full bd-primary-500"> Button </button>
//원하는 사이즈 + full 클래스 동시 사용

<!-- mix type => background-color & border-color -->
<button type="button" class="btn btn-md bg-primary50 bd-primary-500 fc-gray-800">Button</button>
// background와 border를 같이 사용할 경우 color도 같이 입력하여 사용

Chips

  • 칩은 사용자가 선택을 하거나 콘텐츠를 필터링하거나 동작을 트리거하는 데 도움이 됩니다.
  • data-chip-color + chip or chip + [color type] 의 조합으로 이루어져 있습니다.

Option

Type Style Dataset & Class example Description
Dataset primary | secondary | info | success | warning | danger data-chip-color="primary"
class = chip 		컴포넌트의 색상 사용시 chip 클래스 사용 후 dataset에 해당 컬러의 이름 적용
Class background & border & icon class = chip bd-primary100 bg-primary75 컴포넌트 외 컬러 사용시에는 별도의 스타일 클래스 적용

Example

<!-- Chip => only dataset-->
<span data-chip-color="primary" class="chip">Chip</span>
<!-- Chip => only class-->
<span class="chip bd-gray-75 bg-gray-50">Chip</span>
<!-- Chip => dataset + class-->
<span data-chip-color="primary" class="chip bg-primary-600 fc-white-100">Chip</span>

Input fields

  • input field는 화면에 텍스트를 추가 할 수 있으며, 한 줄로 작성 할 수 있는 데이터에 적합합니다.
  • input + [state] 의 조합으로 이루어져 있습니다.

Option

Type State Size Class example Description
text Default [inp-]:
sm | md | lg 		class="input" input의 기본 상태
text Focus [inp-]:
sm | md | lg 		class="input" input의 focus 상태
text Default [inp-]:
sm | md | lg 		class="input inp-success" input의 유효성 검사 통과
text Default [inp-]:
sm | md | lg 		class="input danger" input의 유효성 검사 실패
text Disabled [inp-]:
sm | md | lg 		class="input" disabled input의 비활성화

Example

<!-- Input -->
<input type="text" class="input inp-md" />
<!-- Input => success-->
<input type="text" class="input inp-md success" />
<!-- Input => danger-->
<input type="text" class="input inp-md danger" />
<!-- Input => disabled-->
<input type="text" class="input inp-md" disabled />

Radio & Checkbox & Toggle

  • Radio는 단수 선택시에 사용합니다.
  • Checkbox는 복수 선택 , 토글 기능에 사용합니다.
  • type + [style] | [direction] +[size] 의 조합으로 이루어져 있습니다.
  • direction은 아이콘과 토글에서 사용합니다.
  • size는 아이콘에서만 사용합니다.

Option

Type Style Size State Class example Description
radio [icon] : circle | square
[text] 		[inp-ic-]:
sm | md | lg 		off | on class="icon circle" radio의 스타일 설정
checkbox [icon] : circle | square
[text]
[toggle] 		[inp-ic-]:
sm | md | lg 		off | on class="icon circle" checkbox의 스타일 설정

Example

<!-- radio button -->
<input type="radio" class="icon circle" />
<!-- checkbox -->
<input type="checkbox" class="icon circle" />
<!-- checkbox toggle -->
<input type="checkbox" class="toggle" />
