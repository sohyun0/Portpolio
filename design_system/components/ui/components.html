Design System - Component

Button

  • btn + btn-[size] [type] | [state]의 조합으로 이루어져 있습니다.
  • [size] [type] 은 필수로 지정해야 하고, [state]는 지정하지 않으면 활성화 상태로 적용됩니다.
  • component에 포함되지 않은 스타일은 아래의 방법으로 제작할 수 있고 별도의 상태별 스타일이 필요할 시 atoms.scss에서 작성 부탁드립니다.

Tab menu

  • 탭메뉴는 메뉴간의 이동을 간편하게 할 때 사용합니다. 또한 컨텐츠를 나누어 보여줄때 사용합니다.
  • 탭버튼은 tab-header tab + [style], 버튼의 하위 요소는 data-tab-header 로 표시합니다.
  • 탭컨텐츠는 tab-content, 컨텐츠의 하위 요소는data-tab-content 로 표시합니다.
  • 탭 버튼과 컨텐츠의 데이터 속성값은 동일해야합니다.
  • 탭 내부에 탭을 삽입할 시 tab-style[]클래스에 구분할 수 있도록 클래스 확장 후 사용합니다.

ToolTip

  • 툴팁은 마우스를 오버하거나 클릭할 때 사용자에게 정보를 제공합니다.
  • 탭버튼은 tab-header tab + [style], 버튼의 하위 요소는 data-tab-header 로 표시합니다.
  • 탭컨텐츠는 tab-content, 컨텐츠의 하위 요소는data-tab-content 로 표시합니다.
  • 탭 버튼과 컨텐츠의 데이터 속성값은 동일해야합니다.
  • 탭 내부에 탭을 삽입할 시 tab-style[]클래스에 구분할 수 있도록 클래스 확장 후 사용합니다.

Example

<button type="button" class="tool-btn"">
  <span class="ic ic-md info"></span>
  <div class="tool-tip light" data-dir="top-right">
    <div class="tip"></div>
    Currently, gas prices are outrageously high. It is preferable to perform the transaction after some time.
  </div>
</button>
