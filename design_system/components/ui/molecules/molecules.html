Design System - Component

Dropdown

  • 여러개의 데이터를 한 페이지에 보여줘야 할 때 페이지네이션 기능을 사용하여 다른 데이터를 볼 수 있습니다.
  • 10개 이상의 버튼이 필요할 때 page-ellipsis 클래스를 사용하여 축약 스타일을 사용 할 수 있습니다 .
  • 해당 페이지 활성화 시 active클래스를 사용하여 활성화 스타일을 사용 할 수 있습니다.
  • txt-* 클래스를 사용하여 위치를 변경 할 수 있습니다.

Example

<!-- dropdown -->
<div class="dropdown" tabindex="0">
  <button type="button" class="selected-item">Please Select</button>
  <ul class="option-list scroll">
    <li class="option-item item">Select1</li>
    <li class="option-item item">Select2</li>
  </ul>
</div>

Tab menu

  • 탭메뉴는 메뉴간의 이동을 간편하게 할 때 사용합니다. 또한 컨텐츠를 나누어 보여줄때 사용합니다.
  • 탭버튼은 tab-header tab + [style], 버튼의 하위 요소는 data-tab-header 로 표시합니다.
  • 탭컨텐츠는 tab-content, 컨텐츠의 하위 요소는data-tab-content 로 표시합니다.
  • 탭 버튼과 컨텐츠의 데이터 속성값은 동일해야합니다.
  • 탭 내부에 탭을 삽입할 시 tab-style[]클래스에 구분할 수 있도록 클래스 확장 후 사용합니다.

Option

Type Style Class Dataset State Class Example Description
Button line / bg tab-header data-tab-header active data-tab-header="content1"
class= "tab-header tab-line"
class= "tab-header tab-bg" 		탭메뉴의 버튼 설정
Content - tab-content data-tab-content active data-tab-content="content1"
class= "tab-content tab-line"
class= "tab-content tab-bg" 		탭메뉴의 컨텐츠 설정

Example

<!-- tab menu -->
<div class="tab-line tab-header">
  <button type="button" class="active" data-tab-header="tab01">Tab menu Button 1</button>
  <button type="button" data-tab-header="tab02">Tab menu Button 2</button>
</div>
<div class="tab-line tab-content">
  <div class="active" data-tab-content="tab01">Tab menu Content 1</div>
  <div data-tab-content="tab02">Tab menu Content 2 </div>
</div>
<script>
  tab(".tab-line");
  tab(".tab-line");
</script>
